Lab 9 - JavaScript for the Web



Challenge

Working with your partner, experiment with DOM manipulation.

Problem

The point here is to experiment with the window.document JavaScript object, which corresponds to the web page itself. We're tasked with tagging some elements in our HTML, then manipulating the tagged elements with .getElementByID(), and finally creating and adding new elements as children of our tagged element via JavaScript, with both the .appendChild() and .prepend() methods.
