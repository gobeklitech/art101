Lab 8 - Anonymous Functions & Callbacks


Challenge

Experiment with anonymous functions and callbacks in JavaScript.


Problem

Our task in this lab was to experiment with two features in JavaScript, anonymous functions and Callbacks. Anonymous functions are basically just functions defined ad-hoc. The concept is slippery at first, but we can see how it can save time and energy down the line. Abstractly, callbacks are instructions for the system to follow when an event occurs, such as a button press or user input. In this case, our 'event' was a command to map the array somewhere, and the callback was the function we mapped the array to. Then we got meta with it, and created a function whose input itself is a callback. Far out!
