Lab 15 - AJAX



Challenge

Experiment with jQuery & AJAX

Problem

Here I chose to build a simple Weather application to gain experience interfacing with APIs via AJAX. First, when the user presses 'Activate', we acces the HTML Geolocation API, which attempts to get the latitude and longitude of the user's position. Then, with this data we attempt a call to the Tomorrow.io weather API and try to generate a weather report for the user's location.
