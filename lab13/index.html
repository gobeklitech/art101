Lab 13 - Loops

Challenge

Experiment with loops!

Problem

The task here was to create a form where the user enters an integer. Then, we create a function that tests all the numbers from 1 to the input number for their divisibility by 3, 5 ,7 and 11. For each of those numbers, we assign the words 'Fizz' , 'Buzz' , 'Bang' and 'Bop', respectively. Then, we print the combination of these words depending on the results of the divisibility tests - i.e. if the number is divisible by 3, we see 'Fizz', 3 and 5 we see 'FizzBuzz', etc.
