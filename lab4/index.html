Challenge

The challenge in this lab is to pseudocode, first a real-world task (I chose how to cook the perfect egg), and then a computer task (I chose how to build an image classifier)

Problem

The intention here is to build sequential reasoning skills and to practice outlining a task at a high-level, which will help us when we begin writing programs in JavaScript
