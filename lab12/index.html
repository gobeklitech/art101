Lab 12 - Conditionals

Challenge

Create a function that depends on conditionals.

Problem

The task here was to create a form for the user to input their full name. Then, an event listener attached to a submit button triggers a function that sorts the number of characters in the submission into it's remainder modulo four. This number is then used to determine the user's house, out of Gryffindor, Ravenclaw, Slytherin, and Hufflepuff.
